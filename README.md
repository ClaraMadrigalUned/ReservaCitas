# ReservaCitas

## Descripción

ReservaCitas es una aplicación web desarrollada con ASP.NET MVC 5 y C# para gestionar el registro de citas o reservas. La aplicación permite registrar citas, validar los datos ingresados, consultar la lista de citas registradas, visualizar el detalle de cada cita y filtrar los registros por estado mediante AJAX.

## Instrucciones para ejecutar el proyecto

1. Clonar o descargar el repositorio.
2. Abrir el archivo `ReservaCitas.sln` en Visual Studio 2022 Community o una versión compatible.
3. Esperar a que Visual Studio restaure los paquetes NuGet necesarios.
4. Compilar la solución.
5. Ejecutar el proyecto utilizando IIS Express.
6. Desde la página principal acceder al registro de citas o a la lista de citas mediante el menú de navegación.
